Integrate Unity Catalog functions with DSPy, a framework for building modular AI applications with optimizable pipelines.

## Installation

Install the Unity Catalog DSPy integration from PyPI:

```bash
pip install unitycatalog-dspy
```

For **Databricks Unity Catalog**, install with the optional databricks package:

```bash
pip install unitycatalog-dspy[databricks]
```

## Setup

<Steps>
  <Step title="Create a Unity Catalog Client">
    Initialize the `UnitycatalogFunctionClient`:

    ```python
    from unitycatalog.ai.core.client import UnitycatalogFunctionClient
    from unitycatalog.client import ApiClient, Configuration

    config = Configuration(
        host="http://localhost:8080/api/2.1/unity-catalog"
    )

    api_client = ApiClient(configuration=config)
    uc_client = UnitycatalogFunctionClient(api_client=api_client)

    CATALOG = "my_catalog"
    SCHEMA = "my_schema"
    ```
  </Step>

  <Step title="Create a Unity Catalog Function">
    Define a function for weather data:

    ```python
    def get_weather(city: str) -> str:
        """Retrieve mock weather information for a given city.

        Args:
            city (str): The name of the city to retrieve weather data for.

        Returns:
            str: A string describing the weather for the given city.
        """
        mock_data = {
            "New York": "Sunny, 25°C",
            "Los Angeles": "Cloudy, 20°C",
            "Chicago": "Rainy, 15°C",
            "Houston": "Thunderstorms, 30°C",
            "Phoenix": "Sunny, 35°C",
        }
        return mock_data.get(city, "Weather data not available")

    function_info = uc_client.create_python_function(
        func=get_weather,
        catalog=CATALOG,
        schema=SCHEMA,
        replace=False
    )
    ```
  </Step>

  <Step title="Create UCFunctionToolkit">
    Convert UC functions to DSPy tools:

    ```python
    from unitycatalog.ai.dspy.toolkit import UCFunctionToolkit

    toolkit = UCFunctionToolkit(
        function_names=[f"{CATALOG}.{SCHEMA}.get_weather"],
        client=uc_client
    )

    # Get DSPy-compatible tools
    tools = toolkit.tools
    ```
  </Step>
</Steps>

## Using with DSPy Programs

Integrate UC tools with DSPy's ReAct agent:

```python
import dspy

# Configure DSPy with your LLM
dspy.settings.configure(lm=dspy.LM('openai/gpt-4o'))

# Create a ReAct agent with UC tools
react_agent = dspy.ReAct(
    signature="question -> answer",
    tools=toolkit.tools,
    max_iters=5
)

# Ask questions
result = react_agent(question="What's the weather like in New York?")
print(result.answer)
print("Tool calls made:", result.trajectory)
```

## Testing Tools Directly

Validate tools before integrating with DSPy:

```python
my_tool = toolkit.get_tool(f"{CATALOG}.{SCHEMA}.get_weather")

if my_tool:
    result = my_tool.func(city="New York")
    print(result)
else:
    print("Tool not found")
```

### Create SQL Function

```python
CATALOG = "catalog"
SCHEMA = "schema"

func_name = f"{CATALOG}.{SCHEMA}.get_weather"
sql_body = f"""CREATE OR REPLACE FUNCTION {func_name}(
    city STRING COMMENT 'The name of the city to retrieve weather data for.'
)
RETURNS STRING
LANGUAGE PYTHON
COMMENT 'Retrieve mock weather information for a given city.'
AS $$
    mock_data = {
        "New York": "Sunny, 25°C",
        "Los Angeles": "Cloudy, 20°C",
        "Chicago": "Rainy, 15°C",
        "Houston": "Thunderstorms, 30°C",
        "Phoenix": "Sunny, 35°C"
    }
    return mock_data.get(city, "Weather data not available")
$$
"""

client.create_function(sql_function_body=sql_body)
```

## UCFunctionToolkit API

The `UCFunctionToolkit` converts Unity Catalog functions to DSPy `Tool` instances.

### Parameters

- `function_names` (List[str]): Function names in `catalog.schema.function` format (required)
- `client` (Optional[BaseFunctionClient]): Client for managing functions
- `filter_accessible_functions` (bool): Filter functions by permissions (default: False)

### Properties

- `tools`: Returns list of underlying `dspy.Tool` instances
- `tools_dict`: Dictionary mapping function names to tool wrappers

### Methods

- `get_tool(function_name)`: Get a specific `dspy.Tool` by function name

## Schema Inspection

Examine the generated schema for a tool:

```python
tool = toolkit.get_tool(func_name)

if tool:
    print(f"Tool name: {tool.name}")
    print(f"Tool description: {tool.desc}")
    print(f"Tool arguments: {tool.args}")
    print(f"Tool argument types: {tool.arg_types}")
```

## Features

- **Seamless Integration**: Convert UC functions to DSPy tools with minimal configuration
- **Automatic Schema Generation**: Generate input parameter schemas from UC function definitions
- **Permission Handling**: Built-in support for filtering functions by permissions
- **MLflow Tracing**: Optional MLflow tracing for function execution monitoring
- **Type Safety**: Uses Pydantic v2 with ConfigDict

## Complete Example

<CodeGroup>
```python Full Workflow
import dspy
from unitycatalog.ai.core.client import UnitycatalogFunctionClient
from unitycatalog.ai.dspy.toolkit import UCFunctionToolkit
from unitycatalog.client import ApiClient, Configuration

# 1. Setup client
config = Configuration(host="http://localhost:8080/api/2.1/unity-catalog")
api_client = ApiClient(configuration=config)
uc_client = UnitycatalogFunctionClient(api_client=api_client)

# 2. Create function
def search_database(query: str, limit: int = 10) -> str:
    """Search a database for matching records.
    
    Args:
        query: Search query string
        limit: Maximum number of results to return
    
    Returns:
        JSON string of search results
    """
    # Mock implementation
    return f"Found 5 results for '{query}' (limit: {limit})"

uc_client.create_python_function(
    func=search_database,
    catalog="main",
    schema="tools",
    replace=True
)

# 3. Create toolkit
toolkit = UCFunctionToolkit(
    function_names=["main.tools.search_database"],
    client=uc_client
)

# 4. Configure DSPy
dspy.settings.configure(lm=dspy.LM('openai/gpt-4o'))

# 5. Create ReAct agent
react_agent = dspy.ReAct(
    signature="question -> answer",
    tools=toolkit.tools,
    max_iters=5
)

# 6. Execute
result = react_agent(
    question="Search the database for customers in California"
)

print("Answer:", result.answer)
print("Reasoning:", result.trajectory)
```

```python Multiple Tools
import dspy
from unitycatalog.ai.dspy.toolkit import UCFunctionToolkit

# Define multiple utility functions
def get_weather(city: str) -> str:
    """Get weather for a city."""
    return f"Weather in {city}: Sunny, 22°C"

def calculate_distance(city1: str, city2: str) -> float:
    """Calculate distance between two cities in km."""
    # Mock calculation
    return 450.5

def get_population(city: str) -> int:
    """Get population of a city."""
    populations = {
        "New York": 8336817,
        "Los Angeles": 3979576,
        "Chicago": 2693976
    }
    return populations.get(city, 0)

# Create all functions in UC
for func in [get_weather, calculate_distance, get_population]:
    uc_client.create_python_function(
        func=func,
        catalog="main",
        schema="tools",
        replace=True
    )

# Create toolkit with all tools
toolkit = UCFunctionToolkit(
    function_names=[
        "main.tools.get_weather",
        "main.tools.calculate_distance",
        "main.tools.get_population"
    ],
    client=uc_client
)

# Configure and use
dspy.settings.configure(lm=dspy.LM('openai/gpt-4o'))

react_agent = dspy.ReAct(
    signature="question -> answer",
    tools=toolkit.tools,
    max_iters=10
)

result = react_agent(
    question="What's the weather in New York and how far is it from Chicago?"
)

print(result.answer)
```

## Error Handling

Handle errors gracefully:

```python
try:
    toolkit = UCFunctionToolkit(
        function_names=[func_name],
        client=client
    )
except PermissionError as e:
    print(f"Permission denied: {e}")
except ValueError as e:
    print(f"Invalid configuration: {e}")
```

## Advanced Usage

### Custom Function Execution

```python
tool = toolkit.get_tool(func_name)
if tool:
    result = tool.func(city="New York")
    print(result)
```

### Bulk Function Loading

```python
# Load multiple functions at once
function_names = [
    "catalog.schema.func1",
    "catalog.schema.func2",
    "catalog.schema.func3"
]

toolkit = UCFunctionToolkit(
    function_names=function_names,
    client=client,
    filter_accessible_functions=True  # Only load accessible functions
)

print(f"Loaded {len(toolkit.tools)} tools")
```
