Use functions defined within Unity Catalog (UC) directly as tools within LlamaIndex applications.

## Installation

Install the Unity Catalog LlamaIndex integration from PyPI:

```bash
pip install unitycatalog-llamaindex
```

For **Databricks Unity Catalog**, install with the optional databricks package:

```bash
pip install unitycatalog-llamaindex[databricks]
```

## Setup

<Steps>
  <Step title="Create a Unity Catalog Client">
    Initialize the `UnitycatalogFunctionClient`:

    ```python
    from unitycatalog.ai.core.client import UnitycatalogFunctionClient
    from unitycatalog.client import ApiClient, Configuration

    config = Configuration(
        host="http://localhost:8080/api/2.1/unity-catalog"
    )

    api_client = ApiClient(configuration=config)
    uc_client = UnitycatalogFunctionClient(api_client=api_client)

    CATALOG = "my_catalog"
    SCHEMA = "my_schema"
    ```
  </Step>

  <Step title="Create a Unity Catalog Function">
    Define your function with type hints and docstring:

    ```python
    def add_numbers(a: float, b: float) -> float:
        """
        Adds two numbers and returns the result.

        Args:
            a (float): First number.
            b (float): Second number.

        Returns:
            float: The sum of the two numbers.
        """
        return a + b

    function_info = uc_client.create_python_function(
        func=add_numbers,
        catalog=CATALOG,
        schema=SCHEMA,
        replace=False
    )
    ```
  </Step>

  <Step title="Create UCFunctionToolkit">
    Convert UC functions to LlamaIndex tools:

    ```python
    from unitycatalog.ai.llama_index.toolkit import UCFunctionToolkit

    toolkit = UCFunctionToolkit(
        function_names=[f"{CATALOG}.{SCHEMA}.add_numbers"]
    )

    # Get LlamaIndex-compatible tools
    tools = toolkit.tools
    ```
  </Step>
</Steps>

## Using with LlamaIndex Agents

Integrate UC function tools with a LlamaIndex ReActAgent:

```python
from llama_index.llms.openai import OpenAI
from llama_index.core.agent import ReActAgent

llm = OpenAI(model="gpt-4")

agent = ReActAgent.from_tools(tools, llm=llm, verbose=True)

response = agent.chat(
    "Please call a tool to add 42 and 97 together."
)
print(response)
```

## Direct Tool Testing

Test your tool before integrating with agents:

```python
my_tool = tools[0]

# Call using keyword arguments
result = my_tool.fn(**{"a": 10, "b": 20})
print(result)

# Or use the call API
result = my_tool.call(a=10, b=20)
print(result)
```

### Create SQL Function

Create a Python UDF using SQL syntax:

```python
CATALOG = "catalog"
SCHEMA = "schema"

func_name = f"{CATALOG}.{SCHEMA}.python_exec"
sql_body = f"""CREATE OR REPLACE FUNCTION {func_name}(
    code STRING COMMENT 'Python code to execute. Print final result to stdout.'
)
RETURNS STRING
LANGUAGE PYTHON
COMMENT 'Executes Python code and returns its stdout.'
AS $$
    import sys
    from io import StringIO
    stdout = StringIO()
    sys.stdout = stdout
    exec(code)
    return stdout.getvalue()
$$
"""

client.create_function(sql_function_body=sql_body)
```

## UCFunctionToolkit API

The `UCFunctionToolkit` class converts Unity Catalog functions into LlamaIndex `FunctionTool` instances.

### Parameters

- `function_names` (List[str]): Function names in `catalog.schema.function` format
- `client` (Optional[BaseFunctionClient]): Client for managing functions
- `return_direct` (bool): Whether tools should return output directly (default: False)
- `filter_accessible_functions` (bool): Filter functions by permissions (default: False)

### Properties

- `tools`: Returns list of LlamaIndex `FunctionTool` instances
- `tools_dict`: Dictionary mapping function names to tools

### Return Direct Mode

When `return_direct=True`, the agent returns tool output directly without interpretation:

```python
toolkit = UCFunctionToolkit(
    function_names=[f"{CATALOG}.{SCHEMA}.get_weather"],
    return_direct=True
)
```

## Reserved Arguments

<Note>
LlamaIndex reserves the `properties` argument name. Functions with a `properties` parameter in their input schema cannot be used with LlamaIndex.
</Note>

The toolkit automatically checks for this and raises a `ValueError` if detected.

## Complete Example

<CodeGroup>
```python Full Workflow
from unitycatalog.ai.core.client import UnitycatalogFunctionClient
from unitycatalog.ai.llama_index.toolkit import UCFunctionToolkit
from unitycatalog.client import ApiClient, Configuration
from llama_index.llms.openai import OpenAI
from llama_index.core.agent import ReActAgent

# 1. Setup client
config = Configuration(host="http://localhost:8080/api/2.1/unity-catalog")
api_client = ApiClient(configuration=config)
uc_client = UnitycatalogFunctionClient(api_client=api_client)

# 2. Create function
def calculate_area(length: float, width: float) -> float:
    """Calculate the area of a rectangle.
    
    Args:
        length: Length of the rectangle
        width: Width of the rectangle
    
    Returns:
        Area of the rectangle
    """
    return length * width

function_info = uc_client.create_python_function(
    func=calculate_area,
    catalog="main",
    schema="default",
    replace=True
)

# 3. Create toolkit
toolkit = UCFunctionToolkit(
    function_names=["main.default.calculate_area"]
)

# 4. Create agent
llm = OpenAI(model="gpt-4")
agent = ReActAgent.from_tools(toolkit.tools, llm=llm, verbose=True)

# 5. Execute
response = agent.chat(
    "What is the area of a rectangle that is 12.5 meters long and 8.3 meters wide?"
)
print(response)
```

```python With Multiple Tools
from unitycatalog.ai.llama_index.toolkit import UCFunctionToolkit
from llama_index.core.agent import ReActAgent
from llama_index.llms.openai import OpenAI

# Define multiple functions
def get_weather(city: str) -> str:
    """Get current weather for a city."""
    return f"Weather in {city}: Sunny, 22°C"

def calculate_distance(lat1: float, lon1: float, lat2: float, lon2: float) -> float:
    """Calculate distance between two coordinates."""
    # Simplified calculation
    return ((lat2 - lat1)**2 + (lon2 - lon1)**2)**0.5

# Create functions in UC
for func in [get_weather, calculate_distance]:
    uc_client.create_python_function(
        func=func,
        catalog="main",
        schema="tools",
        replace=True
    )

# Create toolkit with multiple functions
toolkit = UCFunctionToolkit(
    function_names=[
        "main.tools.get_weather",
        "main.tools.calculate_distance"
    ]
)

# Use in agent
llm = OpenAI()
agent = ReActAgent.from_tools(toolkit.tools, llm=llm, verbose=True)

response = agent.chat("What's the weather in San Francisco?")
print(response)
```
</CodeGroup>
