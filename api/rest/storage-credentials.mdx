# Storage Credentials API

Storage credentials contain the authentication information needed to access cloud storage. They are referenced by external locations and used to vend temporary credentials for secure data access.

## Endpoints

### Create Storage Credential

Creates a new storage credential.

```http
POST /api/2.1/unity-catalog/credentials
```

<Note>
  The caller must be a metastore admin to create storage credentials.
</Note>

#### Request Body

<ParamField body="name" type="string" required>
  Name of the storage credential. Must be unique within the metastore.
</ParamField>

<ParamField body="comment" type="string">
  User-provided description.
</ParamField>

<ParamField body="aws_iam_role" type="object">
  AWS IAM role configuration (for AWS storage).
</ParamField>

<ParamField body="azure_service_principal" type="object">
  Azure service principal configuration (for Azure storage).
</ParamField>

<ParamField body="gcp_service_account_key" type="object">
  GCP service account key configuration (for GCP storage).
</ParamField>

#### AWS IAM Role Structure

<ParamField body="role_arn" type="string" required>
  The Amazon Resource Name (ARN) of the AWS IAM role.
</ParamField>

<ParamField body="external_id" type="string">
  External ID for assuming the IAM role.
</ParamField>

#### Azure Service Principal Structure

<ParamField body="directory_id" type="string" required>
  The Azure Active Directory (tenant) ID.
</ParamField>

<ParamField body="application_id" type="string" required>
  The application (client) ID of the service principal.
</ParamField>

<ParamField body="client_secret" type="string" required>
  The client secret for the service principal.
</ParamField>

#### Response

Returns a `CredentialInfo` object with the created credential details.

<ResponseField name="name" type="string">
  Name of the storage credential.
</ResponseField>

<ResponseField name="id" type="string">
  Unique identifier for the credential.
</ResponseField>

<ResponseField name="owner" type="string">
  Username of the credential owner.
</ResponseField>

<ResponseField name="created_at" type="integer">
  Time when created, in epoch milliseconds.
</ResponseField>

<ResponseField name="created_by" type="string">
  Username of the creator.
</ResponseField>

#### Example

<CodeGroup>
```bash AWS Credential
curl -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "aws-prod-credential",
    "comment": "Production AWS access",
    "aws_iam_role": {
      "role_arn": "arn:aws:iam::123456789012:role/uc-access-role"
    }
  }' \
  http://localhost:8080/api/2.1/unity-catalog/credentials
```

```bash Azure Credential
curl -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "azure-prod-credential",
    "comment": "Production Azure access",
    "azure_service_principal": {
      "directory_id": "12345678-1234-1234-1234-123456789012",
      "application_id": "87654321-4321-4321-4321-210987654321",
      "client_secret": "your-client-secret"
    }
  }' \
  http://localhost:8080/api/2.1/unity-catalog/credentials
```

```python Python
from unitycatalog.client import CreateCredential, AwsIamRole

# Create AWS credential
aws_cred = CreateCredential(
    name="aws-prod-credential",
    comment="Production AWS access",
    aws_iam_role=AwsIamRole(
        role_arn="arn:aws:iam::123456789012:role/uc-access-role"
    )
)

credential = await credentials_api.create_credential(aws_cred)
print(f"Created: {credential.name}")
```
</CodeGroup>

### List Storage Credentials

Gets an array of storage credentials from the metastore.

```http
GET /api/2.1/unity-catalog/credentials
```

#### Query Parameters

<ParamField query="max_results" type="integer">
  Maximum number of credentials to return (max 1000).
</ParamField>

<ParamField query="page_token" type="string">
  Opaque pagination token for the next page.
</ParamField>

#### Response

<ResponseField name="credentials" type="array">
  Array of `CredentialInfo` objects.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Opaque token for pagination.
</ResponseField>

#### Example

```bash
curl -H "Authorization: Bearer $TOKEN" \
  http://localhost:8080/api/2.1/unity-catalog/credentials
```

### Get Storage Credential

```http

```

<Note>
  Sensitive fields like secrets are redacted in the response.
</Note>

#### Path Parameters

<ParamField path="name" type="string" required>

</ParamField>

#### Example

```bash
curl -H "Authorization: Bearer $TOKEN" \
  http://localhost:8080/api/2.1/unity-catalog/credentials/aws-prod-credential
```

### Update Storage Credential

Updates a storage credential in the metastore.

```http
PATCH /api/2.1/unity-catalog/credentials/{name}
```

<Warning>
  The caller must be the owner of the credential or a metastore admin.
</Warning>

#### Path Parameters

<ParamField path="name" type="string" required>
  Name of the credential to update.
</ParamField>

#### Request Body

<ParamField body="comment" type="string">
  New comment.
</ParamField>

<ParamField body="owner" type="string">
  New owner username.
</ParamField>

<ParamField body="aws_iam_role" type="object">
  Updated AWS IAM role configuration.
</ParamField>

<ParamField body="azure_service_principal" type="object">
  Updated Azure service principal configuration.
</ParamField>

#### Example

```bash
curl -X PATCH \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "comment": "Updated production AWS credential"
  }' \
  http://localhost:8080/api/2.1/unity-catalog/credentials/aws-prod-credential
```

### Delete Storage Credential

Deletes a storage credential from the metastore.

```http
DELETE /api/2.1/unity-catalog/credentials/{name}
```

<Warning>
  Cannot delete a credential if it's referenced by any external locations.
</Warning>

#### Path Parameters

<ParamField path="name" type="string" required>
  Name of the credential to delete.
</ParamField>

#### Query Parameters

<ParamField query="force" type="boolean">
  Force deletion even if there are dependent external locations.
</ParamField>

#### Example

```bash
curl -X DELETE \
  -H "Authorization: Bearer $TOKEN" \
  http://localhost:8080/api/2.1/unity-catalog/credentials/old-credential
```

## Best Practices

### Use IAM Roles Instead of Access Keys

For AWS, prefer IAM roles over access keys:

```python
# Good: Use IAM role
aws_cred = CreateCredential(
    name="aws-iam-role",
    aws_iam_role=AwsIamRole(
        role_arn="arn:aws:iam::123456789012:role/uc-access-role"
    )
)

# Avoid: Access keys (not supported in UC OSS)
```

### Rotate Credentials Regularly

For Azure service principals, rotate secrets periodically:

```python
# Update credential with new secret
update = UpdateCredential(
    azure_service_principal=AzureServicePrincipal(
        directory_id="...",
        application_id="...",
        client_secret="new-secret-value"
    )
)

await credentials_api.update_credential("azure-prod", update)
```

### Principle of Least Privilege

Grant only necessary permissions to storage credentials:

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "s3:GetObject",
        "s3:ListBucket"
      ],
      "Resource": [
        "arn:aws:s3:::my-bucket",
        "arn:aws:s3:::my-bucket/*"
      ]
    }
  ]
}
```
