    <p className="text-lg dark:text-gray-400 text-gray-600">
      Get up and running with Unity Catalog in minutes
    </p>
  </div>

  <Steps>
    <Step title="Install and Start the Server">
      Clone the repository and start the Unity Catalog server locally.

      ```bash
      git clone https://github.com/unitycatalog/unitycatalog.git
      cd unitycatalog
      bin/start-uc-server
      ```

      The server will start on `http://localhost:8080` with a default catalog and sample data.
    </Step>

    <Step title="Explore with the CLI">
      Use the CLI to list catalogs, schemas, and tables.

      ```bash
      # List all catalogs
      bin/uc catalog list

      # List tables in the default schema
      bin/uc table list --catalog unity --schema default

      # Read table data
      bin/uc table read --full_name unity.default.numbers
      ```

      <Accordion title="Example Output">
        ```
        ┌─────────────┬──────────┬─────────┐
        │    as_int   │ as_double│         │
        ├─────────────┼──────────┼─────────┤
        │      123    │  123.45  │   ...   │
        │      456    │  456.78  │   ...   │
        └─────────────┴──────────┴─────────┘
        ```
      </Accordion>
    </Step>

    <Step title="Query with DuckDB">
      Connect DuckDB to Unity Catalog and start querying your data.

      ```sql
      -- Install extensions
      INSTALL uc_catalog FROM core_nightly;
      LOAD uc_catalog;

      -- Connect to Unity Catalog
      CREATE SECRET (
        TYPE UC,
        TOKEN 'not-used',
        ENDPOINT 'http://127.0.0.1:8080',
        AWS_REGION 'us-east-2'
      );

      -- Attach catalog
      ATTACH 'unity' AS unity (TYPE UC_CATALOG);

      -- Query tables
      SELECT * FROM unity.default.numbers;
      ```
    </Step>

    <Step title="Integrate AI Functions">
      Use Unity Catalog's AI capabilities to manage and call functions in your GenAI applications.

      ```python
      from unitycatalog.ai.core.client import UnitycatalogFunctionClient
      from unitycatalog.client import ApiClient, Configuration

      # Setup client
      config = Configuration(host="http://localhost:8080/api/2.1/unity-catalog")
      client = UnitycatalogFunctionClient(api_client=ApiClient(configuration=config))

      # Create and execute a function
      result = client.execute_function("unity.default.my_function", {"param": "value"})
      ```

      Learn more in the [AI & ML documentation](/ai/introduction).
    </Step>
  </Steps>
