[Daft](https://getdaft.io/) is a library for parallel and distributed processing of multimodal data. It provides a familiar DataFrame API with powerful optimizations for large-scale data processing.

## Installation

Install Daft with Unity Catalog and Delta Lake dependencies:

```bash
pip install -U "daft[unity,deltalake]"
```

## Setup

### Import Required Libraries

```python
import daft
from daft.unity_catalog import UnityCatalog
```

### Start Unity Catalog Server

You need a running Unity Catalog server. For local testing:

```bash
bin/start-uc-server
```

## Connect to Unity Catalog

Create a `UnityCatalog` object to connect Daft to your UC server:

```python
# Connect to local Unity Catalog server
unity = UnityCatalog(
    endpoint="http://127.0.0.1:8080",
    token="not-used",
)
```

<ParamField path="endpoint" type="string" required>
  Unity Catalog server URL
</ParamField>

<ParamField path="token" type="string" required>
  Authentication token (use `"not-used"` for local development)
</ParamField>

### Connect to Databricks Unity Catalog

You can also connect to Unity Catalog in a Databricks workspace:

```python
unity = UnityCatalog(
    endpoint="https://<databricks_workspace_id>.cloud.databricks.com",
    token="<your_databricks_token>",
)
```

## Load and Query Tables

### Load a Table

Load a Unity Catalog table into Daft:

```python
# Load the table metadata
unity_table = unity.load_table("unity.default.numbers")

# Read the table as a Daft DataFrame
df = daft.read_deltalake(unity_table)
```

Daft automatically optimizes operations on Unity Catalog tables:

### Select Columns

```python
# Select specific columns
df_selected = df.select("as_int")
df_selected.show()
```

### Aggregations

```python
# Compute statistics
df.agg(
    df["as_int"].mean().alias("avg_int"),
    df["as_double"].max().alias("max_double"),
    df["as_int"].count().alias("count")
).show()
```

### Complex Transformations

```python
# Chain multiple operations
result = (
    df
    .where(df["as_int"] > 500)
    .select(df["as_int"], df["as_double"])
    .with_column("ratio", df["as_double"] / df["as_int"])
    .sort("ratio", desc=True)
    .limit(10)
)
result.show()
```

## Work with Marksheet Table

```python
# Load marksheet table
marksheet_table = unity.load_table("unity.default.marksheet")
df_marks = daft.read_deltalake(marksheet_table)

# Show students with high marks
df_marks.where(df_marks["marks"] > 800).show()

# Calculate statistics
df_marks.agg(
    df_marks["marks"].mean().alias("average_marks"),
    df_marks["marks"].min().alias("min_marks"),
    df_marks["marks"].max().alias("max_marks")
).show()
```

## Advanced Features

### Predicate Pushdown

Daft automatically pushes filters down to Delta Lake for optimized reads:

```python
# Only scans relevant data files
df_filtered = df.where(
    (df["as_int"] > 500) & (df["as_double"] < 600)
)
```

### Partition Pruning

For partitioned tables, Daft prunes partitions automatically:

```python
# If table is partitioned by date, only scans relevant partitions
df_recent = df.where(df["date"] >= "2024-01-01")
```

### Distributed Processing

Daft can scale to large datasets using distributed compute:

```python
# Set up distributed execution (Ray or multiprocessing)
import daft
daft.context.set_runner_ray()  # Use Ray for distributed processing

# Process large tables efficiently
large_table = unity.load_table("unity.default.large_dataset")
df_large = daft.read_deltalake(large_table)
result = df_large.where(df_large["value"] > 1000).groupby("category").agg(
    df_large["value"].sum().alias("total")
)
result.show()
```

## Integration with Unity Catalog Features

### Credential Vending

Daft automatically receives temporary credentials from Unity Catalog for accessing cloud storage:

```python
# No need to configure S3/Azure/GCS credentials
# Unity Catalog vends them automatically
df = daft.read_deltalake(unity.load_table("unity.default.s3_table"))
```

### Schema Evolution

Daft handles schema evolution in Delta tables:

```python
# Reads tables with evolved schemas
df = daft.read_deltalake(unity.load_table("unity.default.evolving_table"))
```

## Limitations

<Warning>
  - **Read-only**: Daft currently supports reading from Unity Catalog, not writing
  - **Delta Lake only**: Only Delta Lake format tables are supported (not Iceberg or Hudi)
  - **In development**: Daft's Unity Catalog support is under rapid development
</Warning>

## Complete Example

```python
import daft
from daft.unity_catalog import UnityCatalog

# Connect to Unity Catalog
unity = UnityCatalog(
    endpoint="http://127.0.0.1:8080",
    token="not-used",
)

# Explore available tables
print("Available tables:")
for table in unity.list_tables("unity.default"):
    print(f"  - {table}")

# Load and analyze numbers table
numbers_table = unity.load_table("unity.default.numbers")
df_numbers = daft.read_deltalake(numbers_table)

print("\nNumbers table preview:")
df_numbers.show(5)

print("\nFiltered data (as_int > 500):")
df_filtered = df_numbers.where(df_numbers["as_int"] > 500)
df_filtered.show(5)

print("\nStatistics:")
df_numbers.agg(
    df_numbers["as_int"].mean().alias("avg_int"),
    df_numbers["as_int"].max().alias("max_int"),
    df_numbers["as_double"].mean().alias("avg_double")
).show()

# Load and analyze marksheet table
marksheet_table = unity.load_table("unity.default.marksheet")
df_marks = daft.read_deltalake(marksheet_table)

print("\nTop 10 students by marks:")
df_marks.sort("marks", desc=True).limit(10).show()
```
