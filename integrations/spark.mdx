[Apache Spark](http://spark.apache.org) is a multi-language engine for executing data engineering, data science, and machine learning on single-node machines or clusters.

## Why Unity Catalog with Spark?

Integrating Apache Spark with Unity Catalog provides significant advantages:

<CardGroup cols={2}>
  <Card title="Unified Governance" icon="shield">
    Fine-grained access control, automated lineage tracking, and centralized metadata management
  </Card>
  <Card title="Credential Vending" icon="key">
    Securely access data without configuring credentials for each table
  </Card>
  <Card title="Decoupled Storage" icon="layer-group">
    Separate business logic from file paths and storage locations
  </Card>
  <Card title="Multi-Format Support" icon="database">
    Easy access to Delta Lake, Parquet, CSV, and other formats
  </Card>
</CardGroup>

## Prerequisites

<Warning>
  Unity Catalog requires **Apache Spark 3.5.3+** and **Delta Lake 3.2.1+** for full compatibility.
</Warning>

## Download and Install

### Download Apache Spark

Download Apache Spark 3.5.3 or later:

```bash
curl -O https://archive.apache.org/dist/spark/spark-3.5.3/spark-3.5.3-bin-hadoop3.tgz
tar xzf spark-3.5.3-bin-hadoop3.tgz
cd spark-3.5.3-bin-hadoop3
```

Alternatively, [download from the Apache Spark website](https://spark.apache.org/downloads.html).

### Configure Cloud Storage (Optional)

To use Unity Catalog with cloud object storage, configure the Unity Catalog server properties in `etc/conf/server.properties`:

<Tabs>
  <Tab title="AWS S3">
    ```properties
    ## S3 Storage Config
    s3.bucketPath.0=<S3_BUCKET>
    s3.region.0=<S3_REGION>
    s3.awsRoleArn.0=<S3_ROLE>
    # Optional (uses DefaultCredentialsProviderChain if blank)
    s3.accessKey.0=<ACCESS_KEY>
    s3.secretKey.0=<SECRET_KEY>
    ```
  </Tab>
  
  <Tab title="Azure ADLS Gen2">
    ```properties
    ## ADLS Storage Config
    adls.storageAccountName.0=<STORAGE_ACCOUNT_NAME>
    adls.tenantId.0=<TENANT_ID>
    adls.clientId.0=<CLIENT_ID>
    adls.clientSecret.0=<CLIENT_SECRET>
    ```
  </Tab>
  
  <Tab title="Google Cloud Storage">
    ```properties
    ## GCS Storage Config
    gcs.bucketPath.0=<GCS_BUCKET>
    # Optional (uses Application Default Credentials if blank)
    gcs.jsonKeyFilePath.0=/path/to/key.json
    ```
  </Tab>
</Tabs>

### Start Unity Catalog Server

If the UC server isn't already running, start it:

```bash
cd unitycatalog/
bin/start-uc-server
```

## Local Development

Run Spark SQL or PySpark locally against your Unity Catalog server.

<Tabs>
  <Tab title="Spark SQL">
    ```bash
    bin/spark-sql --name "local-uc-test" \
      --master "local[*]" \
      --packages "io.delta:delta-spark_2.13:4.0.0,io.unitycatalog:unitycatalog-spark_2.13:0.3.0" \
      --conf "spark.sql.extensions=io.delta.sql.DeltaSparkSessionExtension" \
      --conf "spark.sql.catalog.unity=io.unitycatalog.spark.UCSingleCatalog" \
      --conf "spark.sql.catalog.unity.uri=http://localhost:8080" \
      --conf "spark.sql.catalog.unity.token=" \
      --conf "spark.sql.defaultCatalog=unity"
    ```
  </Tab>
  
  <Tab title="PySpark">
    ```bash
    bin/pyspark --name "local-uc-test" \
      --master "local[*]" \
      --packages "io.delta:delta-spark_2.13:4.0.0,io.unitycatalog:unitycatalog-spark_2.13:0.3.0" \
      --conf "spark.sql.extensions=io.delta.sql.DeltaSparkSessionExtension" \
      --conf "spark.sql.catalog.unity=io.unitycatalog.spark.UCSingleCatalog" \
      --conf "spark.sql.catalog.unity.uri=http://localhost:8080" \
      --conf "spark.sql.catalog.unity.token=" \
      --conf "spark.sql.defaultCatalog=unity"
    ```
  </Tab>
</Tabs>

<Note>
  The first run may take a few minutes to download dependencies.
</Note>

### Configuration Details

<ParamField path="--packages" type="string" required>
  Delta Spark and Unity Catalog Spark packages
</ParamField>

<ParamField path="spark.sql.catalog.unity.uri" type="string" required>
  Unity Catalog server endpoint
</ParamField>

<ParamField path="spark.sql.catalog.unity.token" type="string">
  Authentication token (empty for local dev, required for production)
</ParamField>

<ParamField path="spark.sql.defaultCatalog" type="string" required>
  Default catalog name to use
</ParamField>

<Info>
  **Naming Convention**: Unity Catalog uses three-part naming: `catalog.schema.table`. You can use either `SELECT * FROM unity.default.marksheet` or `SELECT * FROM default.marksheet` if `unity` is the default catalog.
</Info>

## Query Unity Catalog

### Show Schemas and Tables

<CodeGroup>
```sql Spark SQL
-- Show schemas
SHOW SCHEMAS;

-- Show tables
SHOW TABLES IN default;
```

```python PySpark
# Show tables
spark.sql("SHOW TABLES IN default").show()
```

### Merge Tables

Create a source table and merge it into the target:

<CodeGroup>
```sql Spark SQL
-- Create source table
CREATE TABLE demo.srctable (id INT, desc STRING)
USING delta
LOCATION '/tmp/tables/srctable';

INSERT INTO demo.srctable VALUES (3, "updated");
INSERT INTO demo.srctable VALUES (4, "inserted");

-- Merge operation
MERGE INTO demo.mytable AS target
USING demo.srctable AS source
  ON target.id = source.id
WHEN MATCHED THEN
  UPDATE SET *
WHEN NOT MATCHED THEN
  INSERT *;

-- Check results
SELECT * FROM demo.mytable;
```

```python PySpark
# Create source table
spark.sql("""
  CREATE TABLE demo.srctable (id INT, desc STRING)
  USING delta
  LOCATION '/tmp/tables/srctable'
""")

spark.sql("INSERT INTO demo.srctable VALUES (3, 'updated')")
spark.sql("INSERT INTO demo.srctable VALUES (4, 'inserted')")

# Merge operation
spark.sql("""
  MERGE INTO demo.mytable AS target
  USING demo.srctable AS source
    ON target.id = source.id
  WHEN MATCHED THEN
    UPDATE SET *
  WHEN NOT MATCHED THEN
    INSERT *
""")

# Check results
spark.sql("SELECT * FROM demo.mytable").show()
```
</CodeGroup>

Merge results:
```
+---+--------+
| id|    desc|
+---+--------+
|  1|  test 1|
|  2|  test 2|
|  3| updated|
|  4|inserted|
+---+--------+
```

### Drop Table

<CodeGroup>
```sql Spark SQL
DROP TABLE demo.srctable;
SHOW TABLES IN default;
```

```python PySpark
spark.sql("DROP TABLE demo.srctable")
spark.sql("SHOW TABLES IN default").show()
```
</CodeGroup>

<Warning>
  `DROP TABLE` removes the table from Unity Catalog but does not delete the underlying data files.
</Warning>
